Aplicativo Desafio do Conhecimento

Este aplicativo foi desenvolvido como um incentivo aos alunos de Sistemas de Informação da Unibalsas para a área de programação

Tecnologias Utilizadas

Backend

  • PHP
  • MySQL
  • Framework Laravel
  • WebService

Frontend

  • HTML5
  • jQuery
  • jQueryMobile
  • Json

Dados do Desenvolvedor

Atualmente desenvolve na plataforma web utilizando php com o codeigniter e laravel, plataforma mobile com o Android Nativo e jQuery Mobile. Trabalha como TI atualmente para a empresa Contabilize.

© Rodrigo Macedo

Considerações Finais

Este trabalho visou desenvolver uma aplicação que será útil aos acadêmicos de Sistemas de Informação da Unibalsas. Que esse trabalho venha despertar o interesse para o desenvolvimento mobile.
